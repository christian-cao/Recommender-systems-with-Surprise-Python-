Se aplicaran diferentes métodos en el dataset “MoviLen 1M Dataset” extraído de https://gouplens.org/datasets/movielens/ con el fin de ilustrar los comportamientos de diferentes algoritmos de Machine Learning, enfocados a SISTEMAS DE RECOMENDACIÓN utilizando la librería SURPRISE. Se plantearon diferentes relaciones con las características que conservan la lógica propia para entrenar y predecir con los algoritmos, sin embargo, aplicar procedimientos para mejorar su precisión será motivo de próximas prácticas.
